# Lab 2 - Phân Tích và Trực Quan Hóa Dữ Liệu

## Mô tả
Bài lab này thực hành phân tích và trực quan hóa dữ liệu tuyển sinh đại học sử dụng Python (Pandas, Matplotlib, Scipy).

## Dữ liệu
- **File**: `processed_dulieuxettuyendaihoc.csv`
- Dữ liệu về thí sinh tuyển sinh đại học bao gồm điểm thi, khu vực, giới tính, dân tộc, kết quả xét tuyển,...

## Nội dung

### Phần 1: Sắp xếp và Pivot Table
- Sắp xếp dữ liệu theo điểm thi
- Tạo pivot table thống kê theo các nhóm

### Phần 2: Trình bày Dữ liệu
- Lập bảng tần số và tần suất
- Vẽ biểu đồ cột và biểu đồ tròn
- Mô tả thống kê các biến

### Phần 3: Trực quan hóa theo Nhóm
- Biểu đồ phân loại theo giới tính, khối thi, khu vực
- Phân tích kết quả đậu/rớt theo nhóm

### Phần 4: Trực quan hóa Nâng cao
- Biểu đồ đường (Line plot)
- Tạo biến phân loại
- Biểu đồ Multiple Line và Drop-line

### Phần 5: Phân phối và Tương quan
- Mô tả phân phối (Box-plot, Histogram, QQ-plot)
- Phân tích tương quan giữa các biến
- Scatter plot và Scatter matrix

## Công cụ
- Pandas
- NumPy
- Matplotlib
- Scipy
